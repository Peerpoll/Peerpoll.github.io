Snakemake学习

2019-07-12

Snakemake Rule rule sort(名字): input: 'xxx.txt'#(输入文件的文件名) output: 'xxxxx.txt'##(处理后的结果文件名) shell: 'sort {input}>{output}' ###相应需要运行的代码 python(run:) R(run:R())

建议安装anaconda，方便使用

利用anaconda创建一个专属环境

conda create -n snakemake python=3*

激活环境

source activate snakemake

安装snakemake 建议先安装 mamba 能够快速解决环境问题

conda install mamba -c conda-forge
再安装snakemake
mamba install snakemake

关闭环境

source deactivate

基本方法是这样的，根据需要编写自己的代码脚本

写完代码脚本之后 运行snakemake脚本

snakemake -s snakefile.py

断点运行

snakemake -s snakefile.py --rerun-incomplete

主要参考孟叔的使用SnakeMake搭建生信流程
