python数据处理常使用的模块及介绍

2019-07-12 python

  1. numpy 高效的处理数据、提供数组支持，是许多模块的基础，如：pandas、scipy、matplotlib
  2. pandas 主要用于进行数据探索和数据分析
  3. matplotlib 用于作图，进行可视化
  4. scipy 进行数值计算，支持矩阵运算，还有许多高等数据处理，比如积分、傅里叶变换、微分方程求解
  5. statsmodels 主要用于统计分析
  6. Gensim 用于文本挖掘
  7. sklearn 用于机器学习
  8. keras 用于深度学习
